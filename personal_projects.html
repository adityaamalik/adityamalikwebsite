My personal projects have mostly revolved around Full Stack Web Applications using MERN stack. I try to build products which have some real life application also. Here is a given list of projects that I have built over the course of my coding journey.


Varilla PC Care

Varilla PC Care is a desktop application built using Electron JS framework. It is a full fledged PC Care application which has tons of features. It gives insights on CPU, GPU and RAM usage. It also gives network insights, redundant files insights, and other threats to the computer. It also gives desktop notifications, remove junk files, provides a secure vault for storing your passwords.


Docsrecord

Docsrecord is a website for doctors to store their patient records and manage appointments. It is a generic website which can be used by anyone. It has features like E-Mail notifications, Razporpay payment gateway and Google OAuth for Authentication.


Dummy Cart E-Commerce App

Dummy Cart is a full fledged E-commerce web application which has tons of features. It is built using ReactJS, Styled Components, Ant Design Library in the frontend and NodeJS, Express and MongoDB in the backend. It has features like E-mail subscription list, Razorpay payment gateway, Multiple products inside multiple categories, checkout feature and add to cart feature. It also has an Admin Panel which can be used to manage the store front such as Add/Delete/Update Products, Categories, Orders, Email Subscriptions, etc.


Midnight Soiree Fashion Store

Midnight Soiree fashion store is another E-commerce store which has very beautiful UI. It uses NodeJS + Express for the backend combined with the EJS templating engine. It has features like Add to Cart, Checkout, Payment Gateway, etc.
